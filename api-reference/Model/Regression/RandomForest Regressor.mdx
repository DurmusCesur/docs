title: 'RandomForest Regressor'
description: '
Random Forest Regressor creates an ensemble of decision trees, each trained on a random subset of the data. The final prediction is the average of all tree outputs, producing robust and stable results.


It is one of the most reliable and versatile regression methods, requiring minimal parameter tuning.
'
---

<Note>
Random Forest provides built-in **feature importance** rankings, helping you understand which input features contribute most to the prediction. More trees generally produce better results — `100–300` is a good starting range.

</Note>


### Use Cases

- **Complex geometric predictions**: Predict properties with non-linear relationships (e.g., structural load from multiple dimensional features).
- **Default regression choice**: When unsure which algorithm to use, Random Forest is an excellent and safe default.
- **Feature importance analysis**: Discover which geometric features have the most influence on the predicted value.
