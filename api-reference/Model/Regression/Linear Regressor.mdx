title: 'Linear Regressor'
description: '
Linear Regression is the most fundamental regression algorithm. It models the relationship between features and a target value by fitting a straight line (or hyperplane) that minimizes the sum of squared errors.


It is the simplest, fastest, and most interpretable regression model — always a good starting point.
'
---

<Note>
Linear Regression has **no hyperparameters** to tune, making it extremely easy to use. After training, each coefficient directly tells you how much the target changes when that feature increases by one unit. Always try Linear Regression first as a baseline.

</Note>


### Use Cases

- **Simple geometric relationships**: Predict one geometric property from others when the relationship is approximately linear (e.g., area from perimeter).
- **Baseline model**: Establish a performance baseline before trying more complex models.
- **Trend analysis**: Understand how geometric features relate to each other (e.g., how building height relates to floor area).
